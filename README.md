# Restaurant Website

A simple restaurant website built with HTML and Tailwind CSS, designed with a clean and modern interface.
It includes four main pages: Home, Menu, About, and Contact.
The Menu page showcases food cards, while the Contact page features a form, opening hours, and a Google Maps embed.
Minimal JavaScript is used to handle basic interactivity.
The project focuses on responsive design, layout structure, and user experience.
Tailwind CSS (CLI) is used for styling and consistency across pages.
